About

I started this project as a way to do anything you can imagine to organize music. You can use a freeform workspace to find any song, then its up to the user what to do after that.Each element is added to expand the limits of what each user can do, and each element has its own seperate page if you want a deeper dive with one particular concept.

Note: this is run using free api keys and therefore have usage limits, seach results will work for a long while, but not forever.
