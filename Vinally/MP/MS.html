THIS IS HOW YOU USE THE SANDBOX

  1. Right click Anywhere on the page, to create the right click menu
  2. You can use the Right click menu or the buttons at the bottom of the page to begin creating an element

      3. Song Card

    1. Begin by entering the name of any song into the search bar then click search
    2. Three songs from both Youtube and Spotify will apear

      *you can use the scrollwheel to see other cards in each pile

    3. Select the songs you want on the page by clicking on them
    4. You can see all of the songs you select on the bottom of the menu, scroll if you cant see them all
    5. Press Submit Songs to place it on the page

      TierList

    1. In a tier list you can rank things, usually by letter grades
    2. If you want to add another tier to the TierList click “add tier”
    3. If you want to remove a tier click the button with the x through it
    4. If you want to name the tier click on the name and type the name
    5. When you are done with your TierList click “create tier” to place on the page.

      Tournament

    1. Select one of the presented options (2,4,8,16)

      * this will decide how many songs you have space for

    2. Click “Submit” to place on the page

      Container

    1. Selected the title of the container and type to name it
    2. When done click “Submit” to place on page

  3. Each element has different behavior when on the page, one universal rule is dragging
  4. If you click & hold on an element then move your mouse any element will follow your cursor
  5. when you release your cursor the element will be placed at the cursor, or placed inside another element if applicable
  6. If you want to delete an element right click an element and select "delete"

      7. Song Card

    1. Can be dragged into the three other elements

      TierList

    1. Song Cards can be dragged into each tier
    2. Song Cards can be dragged into any position within their respective tier
    3. If there are too many song cards to see you can scroll in each tier row

      Tournament

    1. Song Cards can be dragged into each slot

      * recommended filling the slots on the outside only

    2. When two slots next to each other are filled, double click to select a winner
    3. Continue until you have a final winner

      Container

    1. Song Cards can be dragged inside
